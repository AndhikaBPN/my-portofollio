# Andhika Bagaskara - Portfolio

Personal portfolio website for Andhika Bagaskara Putra Nusantara, a QA/QC Engineer with 3+ years of experience in manual testing, API testing, and automation testing for web & mobile applications.

## Preview

A static portfolio website built with pure HTML, CSS, and JavaScript. Dark minimalist design with navy/slate color scheme.

## Tech Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)

## Features

- Responsive design (mobile, tablet, desktop)
- Dark minimalist theme (no pure black)
- Smooth scroll animations
- Project filter (All, Web, Mobile, Automation)
- Mobile navigation toggle
- Downloadable CV

## Sections

- **Hero** — Introduction with profile photo and download CV button
- **About** — Summary and quick stats
- **Projects** — 9 project cards with filter functionality
- **Experience** — Work history timeline
- **Tech Stack** — Skills and tools organized by category
- **Education** — Academic background
- **Contact** — Email, phone, LinkedIn, location

## How to Run

Simply open `index.html` in any modern browser. No build tools or dependencies required.

```bash
# Or use a local server (optional)
npx serve .
```
